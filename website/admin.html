Publish to GitHub

To update your live website, follow these steps:

  1. Export your content by clicking the button below. This downloads a content.json file with all your changes.
  2. Replace the existing content.json file in your website repository with the downloaded file.
  3. Commit and push the changes to your GitHub repository.
  4. If using GitHub Pages, the changes will deploy automatically.
git add content.json
git commit -m "Update website content"
git push origin main
